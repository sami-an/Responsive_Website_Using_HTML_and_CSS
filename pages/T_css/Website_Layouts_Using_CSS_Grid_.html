1- "grid" The grid property is a shorthand property for:
2- "gap" The gap property defines the size of the gap between the rows and columns .
3- "grid-template-rows" specifies the number and the heights of the rows in a grid layout.
4- "grid-template-columns" specifies the number and the widths of columns in a grid layout.
5- "grid-template-areas" specifies areas within the grid layout.
6- "grid-auto-rows" sets a size for the rows in a grid container.This property affects only rows with the size not set.
6- "grid-auto-columns" sets a size for the columns in a grid container.
7- "grid-auto-flow" controls how auto-placed items get inserted in the grid.
8- " grid-row-start: 2;" defines on which row-line the item will start.
8- " grid-row-end: span 3;" defines on which row-line the item will start.
